## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/jolbol1/jolbol1.github.io.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your Github Login and Github Personal Access Token in a new `.env` file
   ```sh
   GITHUB_LOGIN="";
   GITHUB_PERSONAL_ACCESS_TOKEN=""
   ```
4. Spin it up!
   ```sh
   npm run develop
   ```
